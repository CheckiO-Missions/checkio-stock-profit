Ти брокер на фондовій біржі. Вирішив зробити лише одну повну операцію: придбати акцію та продати її пізніше, щоб отримати прибуток. "Коротка позиція" (спочатку продати, потім викупити) на цьому ринку заборонена.

Ціни купівлі та продажу у кожен конкретний момент однакові (у кожен момент ти можеш купити акцію за ціною x або продати наявну акцію за цією ж ціною x) і показані в даному {% if is_js %}масиві{% else %}списку{% endif %} stock.

Отже, ти повинен обрати найкращі ціни на придбання акції та її пізнішого продажу, щоб отримати максимальний прибуток від операції. Твоя функція повинна повернути цей максимально можливий прибуток. Якщо неможливо отримати прибуток із заданими цінами (він <= 0), твоя функція повинна повернути 0.

Вхідні дані: Ціна акції як {% if is_js %}масив{% else %}список{% endif %} цілих чисел.

Вихідні дані: Максимально можливий прибуток як ціле число.
