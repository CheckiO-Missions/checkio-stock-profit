Als Börsenmakler, der sich auf eine einzelne Transaktion konzentriert, versuchen Sie, Ihren Gewinn zu maximieren, indem Sie eine Aktie zu einem niedrigeren Preis kaufen und sie später zu einem höheren Preis innerhalb eines bestimmten Zeitraums verkaufen.

Ihre Funktion muss die Kursschwankungen der Aktie im Laufe der Zeit analysieren. Sie sollte die profitabelste Gelegenheit ermitteln, indem sie den maximal möglichen Gewinn berechnet, der innerhalb dieser Schwankungen erzielt werden kann. Das bedeutet, dass Sie den höchsten Preis finden müssen, zu dem Sie die Aktie verkaufen können, nachdem Sie sie zum niedrigstmöglichen Preis gekauft haben.

Wenn es jedoch keine Möglichkeit gibt, einen Gewinn zu erzielen - z.B. wenn der Aktienkurs konstant fällt oder gleich bleibt - sollte die Funktion den Wert Null zurückgeben, was bedeutet, dass es keine realistische Möglichkeit für eine profitable Transaktion gibt.
